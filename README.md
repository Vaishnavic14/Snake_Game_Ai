# Goal Description
Build a classic Snake game using HTML, CSS, and JavaScript. The game will feature a responsive design, score tracking, and smooth controls.

User Review Required
None.

# Proposed Changes
Snake Game
[NEW] 
index.html
Main entry point for the game.
Contains the canvas element and UI for score.

[NEW] 
style.css
Styling for the game board, snake, food, and UI.
Modern, dark-themed aesthetic.

[NEW] 
script.js
Game logic: movement, collision detection, score keeping, game loop.

# Verification Plan
Automated Tests
None (Visual game).
Manual Verification
Open index.html in a browser.
Verify the snake moves with arrow keys.
Verify eating food increases score and snake length.
Verify collision with walls or self ends the game.
